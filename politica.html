POLITICA DE PRIVACIDAD

Nos damos cuenta de que la privacidad de su información personal es importante. Esta política describe qué información se puede recopilar cuando utilice nuestra aplicación móvil, cómo se utiliza esa información y cómo se mantiene dicha información.

Nunca vendemos, distribuimos o revelamos información de identificación personal a terceros que no sean de aplicación de la ley.

La información general que recopilamos es en la mayoría de los casos anonimizada y utilizada puramente con propósitos estadísticos o analíticos. Esta información no se puede vincular a usted como un individuo a menos que se indique lo contrario.

Cuando utilice nuestra aplicación, está dando su consentimiento a las prácticas descritas en esta política.

¿Qué son las aplicaciones?

Las aplicaciones son archivos de información que contienen contenido de texto ilimitado que se almacena en las computadoras del usuario para mejorar el rendimiento del sitio web. Estos archivos permiten reconocer al usuario y adaptar el sitio web a sus preferencias. Las aplicaciones suelen contener el nombre del sitio web al que pertenecen, la hora en que se almacenan en la computadora del usuario y un número único.

¿Para qué se utilizan las aplicaciones?

Las aplicaciones se utilizan para personalizar el contenido del sitio web o la aplicación móvil según las preferencias del usuario individual y optimizar el trabajo con el sitio web especificado. También se utilizan para recopilar estadísticas generales anónimas que nos ayudan a comprender cómo trabaja el usuario con los sitios web, ayudando a mejorar su estructura y contenido sin tener acceso a la información personal del usuario.

¿Las aplicaciones contienen información personal?

La información personal que se acumula a través de las aplicaciones solo se puede utilizar para realizar determinadas operaciones del usuario. Dicha información está encriptada de una manera que hace que personas no autorizadas accedan a ella.

Las aplicaciones desarrolladas no contienen información personal ni están vinculadas a una cuenta, ya que no se permite la creación de cuentas dentro de la aplicación.

Eliminar las aplicaciones

Como regla general, las aplicaciones utilizadas para la navegación web y las aplicaciones móviles le permiten guardar aplicaciones de forma predeterminada. Esta configuración se puede cambiar para que el almacenamiento automático de aplicaciones se bloquee en el navegador web o se informe al usuario cada vez antes de que las cookies se almacenen en la computadora. Puede encontrar información detallada sobre las diferentes formas de trabajar con las aplicaciones en la configuración de la aplicación del navegador web.

Una vez eliminada la aplicación todos los datos y cuentas asociados son eliminados al residir únicamente en el terminal del usuario.

La restricción de aplicaciones puede afectar algunas funciones del sitio web y la aplicación móvil.
